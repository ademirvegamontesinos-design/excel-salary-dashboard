# Dashboard de Análisis Salarial en Ciencia de Datos

## Descripción General

Dashboard interactivo desarrollado en Microsoft Excel para analizar salarios en puestos relacionados con Ciencia de Datos. Permite explorar información por cargo, país y tipo de empleo mediante filtros interactivos y visualizaciones dinámicas.

## Herramientas Utilizadas

- Microsoft Excel
- Tablas Dinámicas
- Validación de Datos
- Fórmulas y Funciones
- Visualización de Datos
- Segmentadores de Datos (Slicers)

## Vista Previa del Dashboard

![Vista Previa del Dashboard](dashboard_demo_hd.gif)

## Características

- Filtrado por puesto de trabajo
- Filtrado por país
- Filtrado por tipo de empleo
- Visualización geográfica de salarios
- Comparación de salarios por cargo
- Indicadores clave de desempeño (KPIs)
